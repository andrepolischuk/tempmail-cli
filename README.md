# tempmail-cli [![Build Status][travis-image]][travis-url]

> CLI wrapper for [temp-mail.ru][temp-mail]

## Install

```sh
npm install --global tempmail-cli
```

## Usage

```sh
tempmail --help

  CLI wrapper for temp-mail.ru

  Usage
    tempmail

  Options
    --create, -c    Generate new email
    --get-mail, -g  Get messages

  Examples
    tempmail
    tempmail --get-mail
```
